# Chatbox plugin
To use this chatbox plugin, you have to add this to <strong>index.html</strong>: <br>
```html
<body>
  ...
  <script src="https://cdn.jsdelivr.net/gh/GiangHoGoVap/chatbox_plugin_demo/dist/widget.js"></script>
</body>
```
